unite-webcolorname
==================
This is a unite source plugin.

DESCRIPTION
-----------
unite-webcolorname is a unite source plugin which provides "Web Color Names".

FEATURES
--------
1. Lists "Web Color Names" colorfully.
2. Insert/Yank color name, "#rrggbb", "rgb(r,g,b)".
